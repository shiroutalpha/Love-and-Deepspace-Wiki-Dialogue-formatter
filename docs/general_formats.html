Script Block

Wrap your script with <Script> to add {{Script/start}} and {{Script/end}}.

  • If you leave a key out, the template just falls back to its default value (e.g., Toggle defaults to true, Hide Title defaults to false, and Scheme just matches your wiki's theme and script theme).
  • If you do include a key, make sure its value follows the format below. Also, you don't have to list every single one—anything you skip will just be set to default anyway.
#KeyValid Values (if included)Description
1TitleAny textAdds a title to the script.
2Hide TitleYes for true or No for falseHides the title if set to Yes.
3ToggleYes for true, No for false, or CollapsedControls whether the script is collapsible.
4SchemeDark or LightForces a color scheme; if omitted, the wiki theme is used.
5ThemeAny textSets a custom theme name for event scripts.
6Previous ChaptersLink 1,, Display 1;; Link 2,, Display 2Adds previous chapter navigation links.
7Next ChaptersLink 1,, Display 1;; Link 2,, Display 2Adds next chapter navigation links.
8Edit ScriptAny text (URL)Overrides the default edit link.

Simple <Script>

<Script>
Rafayel: Hello
Result ▼
{{Script/start}}
{{Script/dialogue|Rafayel|Hello}}
{{Script/end}}

With keys

<Script
Title: Birthday 2026
Toggle: No
Scheme: Light
Theme: Birthday>
Rafayel: Hello
Result ▼
{{Script/start|Title=Birthday 2026|Toggle=false|Scheme=Light|Theme=Birthday}}
{{Script/dialogue|Rafayel|Hello}}
{{Script/end}}
Important: The Hide Title, Toggle and Scheme value is strictly validated. Any other value will cause the entire script block to be marked as invalid and highlighted in red.

Dialogue

Basic dialogue

Character: Dialogue → {{Script/dialogue|Character|Dialogue}}

Rafayel: Hello
Result ▼
{{Script/dialogue|Rafayel|Hello}}

Dialogue with expression

Character (Expression): Dialogue → {{Script/dialogue|Character|Dialogue|Expression=Expression}}

Rafayel (smiling): Hello
Result ▼
{{Script/dialogue|Rafayel|Hello|Expression=smiling}}

Dialogue with character override

Character - Character Override: Dialogue → {{Script/dialogue|Character|Dialogue|Character Override=Character Override}}

Rafayel - ???: Hello
Result ▼
{{Script/dialogue|Rafayel|Hello|Character Override=???}}

Character - Character Override (Expression): Dialogue → {{Script/dialogue|Character|Dialogue|Expression=Expression|Character Override=Character Override}}

Rafayel - ??? (smiling): Hello
Result ▼
{{Script/dialogue|Rafayel|Hello|Expression=smiling|Character Override=???}}

Note: The override is useful when you want to display a different character name in the script.

Full Parameter Reference

The {{Script/dialogue}} template supports the following parameters. The converter automatically generates Character, Dialogue, Character Override, and Expression. All other parameters must be added manually to the output.

Parameter Description Auto-generated?
Character Fills in the name of the Character that is speaking. If the input is BLANK, no name tag will be used. ✅ Yes
Dialogue Displays the dialogue text. You can use ;; to add multiple lines under the same name tag (for uninterrupted chains). ✅ Yes
Character Override Overrides the displayed character name while keeping the original character's color-coding (e.g., Sylus → ???). ✅ Yes
Expression Switches between different expressions for Avatars. ✅ Yes
Text Bubble Changes the display from a text box to a text bubble. ❌ Manual
Avatar Adds an Avatar next to the text. Check available Avatars here. ❌ Manual
Circle Set to true to show the Avatar as a circle (used for phone Avatars). ❌ Manual
Avatar Width Resizes the Avatar when necessary. ❌ Manual
Text Background Changes the background color of the Text Box/Bubble. ❌ Manual
Text Border Changes the styling of the border around the Text Box/Bubble. ❌ Manual
Text Color Changes the color of the text within the Text Box/Bubble. ❌ Manual
Align Changes the alignment of the dialogue box. Default: left for all characters, right for Main Character. Supports Left, Center, Right. ❌ Manual
Align Text Aligns the text within the dialogue box. Supports left, center, right. ❌ Manual

Example of manual addition: If you want to add an Avatar and center alignment, edit the output from {{Script/dialogue|Rafayel|Hello}} to {{Script/dialogue|Rafayel|Hello|Avatar=Rafayel|Align=Center}}.

Heading

Use ## to create a heading. You can optionally add a subtitle inside <...>.

Basic heading

## Text → {{Script/heading|Text}}

## Introduction
Result ▼
{{Script/heading|Introduction}}

Heading with subtitle

## Text <Subtitle> → {{Script/heading|Text|Subtitle=Subtitle}}

## Scene 1 <The Beginning>
Result ▼
{{Script/heading|Scene 1|Subtitle=The Beginning}}

Note: To add custom alignment, you can manually edit the output to add |Align Text=....

Narration

Use > for standard narration or >> for large narration blocks (supports multiple lines).

Single-line narration

> Text → {{Script/narration|Text}}

> The room was silent.
Result ▼
{{Script/narration|The room was silent.}}

Single-line large narration

>> Text → {{Script/narration large|Text}}

>> The wind howled.
Result ▼
{{Script/narration large|The wind howled.}}

Multi-line large narration

>> Text1 followed by >> Text2 on subsequent lines will be combined into one block.

>> The wind howled.
>> The storm approached.
Result ▼
{{Script/narration large|The wind howled.;;The storm approached.}}

The converter automatically joins multiple >> lines with ;;. The block closes when a different format (dialogue, heading, etc.) is encountered.

Note: To add custom alignment, you can manually edit the output to add |Align Text=....

Quote

Use ^ to insert a quote, optionally with a source.

Basic quote

^Text → {{Script/quote|Text}}

^To be, or not to be.
Result ▼
{{Script/quote|To be, or not to be.}}

Quote with sources

^Text <Source> → {{Script/quote|Text|Source}}

^That's one small step for man. <Neil Armstrong>
Result ▼
{{Script/quote|That's one small step for man.|Neil Armstrong}}

Note: To add custom alignment, you can manually edit the output to add |Align Text=....

Script Type

Use these tags to change script style.

  • <type/Radio> → {{Script/type|Radio}}
  • <type/Moment> → {{Script/type|Moment}}
  • <type/Text Message> → {{Script/type|Text Message}}
  • <type/Reset> → {{Script/type|Reset}}
<type/Radio>
Result ▼
{{Script/type|Radio}}

Use <type/Reset> to reset the script style

<type/Reset>
Result ▼
{{Script/type|Reset}}
