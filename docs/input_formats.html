Choice (Tabbed Menu)

Use <choice> and </choice> to create a tabbed choice menu. Inside, each tab is introduced with ? Title.

  • <choice> – start of choice block
  • ? Choice Title – tab title, followed by its content (any valid script lines)
  • </choice> – end of choice block

Basic choice

<choice>
? Help Rafayel
Rafayel: Thank you for helping me!
Zayne: That was a wise choice.
? Ignore Rafayel
Rafayel: Why didn't you help me?
Sylus: That was a mistake.
</choice>
Result ▼
<tabber>
|-|Help Rafayel =
{{Script/dialogue|Rafayel|Thank you for helping me!}}
{{Script/dialogue|Zayne|That was a wise choice.}}
|-|Ignore Rafayel =
{{Script/dialogue|Rafayel|Why didn't you help me?}}
{{Script/dialogue|Sylus|That was a mistake.}}
</tabber>

Nested choices

Choices can be nested inside other choices, phones, or any block.

<choice>
? Outer option
Xavier: This is outer.
<choice>
? Inner A
Zayne: Inner A.
? Inner B
Rafayel: Inner B.
</choice>
? Another outer
Sylus: Another outer.
</choice>
Result ▼
<tabber>
|-|Outer option =
{{Script/dialogue|Xavier|This is outer.}}
<tabber>
|-|Inner A =
{{Script/dialogue|Zayne|Inner A.}}
|-|Inner B =
{{Script/dialogue|Rafayel|Inner B.}}
</tabber>
|-|Another outer =
{{Script/dialogue|Sylus|Another outer.}}
</tabber>
Note: All content inside a tab is processed recursively, so you can freely mix dialogues, headings, narrations, interactions, and even phone blocks inside a choice tab.

Interaction

Use !! to create interactive elements. Format: !! Type - Text <optional keys>.

  • Type – name of the interaction type (e.g., Click, Tap, Swipe, Drag, Arrow)
  • Text – adds text below the interaction icon (optional)
  • optional keys (optional) – inside < >:
    • Flip → Flip=true
    • Rotate number (1–360) → Rotate=rotate number
    • Rotate number//Flip → Rotate=rotate number|Flip=true

Basic interaction

!! Type → {{Script/interaction|Type}}
!! Arrow
Result ▼
{{Script/interaction|Arrow}}

Interaction with text

!! Type - Text → {{Script/interaction|Type|Text}}
!! Tap - Tap to continue
Result ▼
{{Script/interaction|Tap|Tap to continue}}

Interaction with rotate

!! Type <Rotate number> → {{Script/interaction|Type|Rotate=Rotate number}}
!! Swipe - Swipe left <90>
Result ▼
{{Script/interaction|Swipe|Swipe left|Rotate=90}}

Interaction with flip

!! Type <Flip> → {{Script/interaction|Type|Flip=true}}
!! Arrow Curve - Drag object <flip>
Result ▼
{{Script/interaction|Arrow Curve|Drag object|Flip=true}}

Interaction with rotate and flip

!! Type <Rotate number//Flip> → {{Script/interaction|Type|Rotate=Rotate number|Flip=true}}
!! Tap - Tap here <45//flip>
Result ▼
{{Script/interaction|Tap|Tap here|Rotate=45|Flip=true}}
