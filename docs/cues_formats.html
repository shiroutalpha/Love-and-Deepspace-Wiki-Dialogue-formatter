Context

Use <Context> to insert a context block. You can optionally specify a type, currently supports Title and Instructions.

Basic context

<Context> Text → {{Script/context|Text}}
<Context> This is a context note.
Result ▼
{{Script/context|This is a context note.}}

Context with type

<Context Type> Text → {{Script/context|Type=Type|Text}}
<Context Title> Important information
Result ▼
{{Script/context|Type=Title|Important information}}
<Context Instructions> Follow these steps
Result ▼
{{Script/context|Type=Instructions|Follow these steps}}
